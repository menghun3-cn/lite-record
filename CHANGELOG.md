# Changelog

本项目的所有重要变更均记录于此文件。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [0.1.0] - 2026-05-29

首个公开版本。

### 新增

- 支持录制整个桌面或指定窗口
- 全局快捷键：`Ctrl+Shift+R` 开始录制，`Ctrl+Shift+S` 停止录制
- 开始录制前 3 秒倒计时，倒计时期间可取消
- 主窗口最小化/不可见时，快捷键开始会先恢复主窗口
- 系统托盘常驻，显示录制状态；双击托盘或菜单「显示主窗口」可恢复窗口
- 关闭主窗口时隐藏到托盘（应用不退出）
- 录制时显示红色边框提示，主窗口自动最小化
- 录制时长实时显示，停止后展示输出文件名
- 视频保存至 `%USERPROFILE%\.lite-record\video`，支持一键打开目录
- 单实例运行：重复启动时聚焦已有窗口
- 应用内错误提示条（替代原生弹窗，避免最小化时无法关闭）

### 修复

- 修复录制开始/停止并发导致的状态不一致与文件占用问题
- 修复托盘恢复主窗口无效的问题（Windows 主线程派发）
- 修复录制状态文案显示不正确的问题

### 技术说明

- 视频格式：MP4 / H.264 / 30fps
- 平台：Windows 10/11（64 位）
- 依赖：Microsoft Edge WebView2 运行时（系统通常已预装）

### 已知限制

- 仅支持 Windows
- 不支持系统音频与麦克风录制
- 不支持自定义保存路径、码率与快捷键
- 不支持区域录制与录制历史列表
